# 🔮 Mystic Tarot · Мистическое Таро

Мини-приложение Таро с 78 уникальными SVG-картами, раскладами, анализом совместимости и психологическим архетипическим портретом. Готово для публикации как VK Mini App.

## ✨ Возможности

### 🃏 78 карт с уникальными SVG-иллюстрациями
- **22 старших аркана** — каждый с собственной художественной иллюстрацией (Шут, Маг, Жрица, Башня, Звезда, Мир и др.)
- **56 младших арканов** — Жезлы (Огонь 🜂), Кубки (Вода 🜄), Мечи (Воздух 🜁), Пентакли (Земля 🜃)
- **Красивая рубашка** с пентаграммой, солнцем, луной и глазом
- Полные интерпретации для прямого и перевёрнутого положения

### 📚 Разделы приложения

| Раздел | Описание |
|--------|----------|
| **☀️ Карта дня** | Ежедневная вытяжка с детальной интерпретацией (любовь, карьера, психология) |
| **📚 Расклады** | 3 карты (прошлое/настоящее/будущее), Кельтский крест (10 позиций), Да/Нет |
| **❤️ Совместимость** | Парный расклад с анализом 10 комбинаций стихий, процентом совместимости, советом паре |
| **🧠 Психология** | 5-карточный архетипический портрет (текущее состояние, внутренний ребёнок, внутренний взрослый, тень, духовная задача) |
| **📜 История** | Все расклады сохраняются в localStorage с возможностью просмотра и удаления |

### 🎨 Дизайн
- **Мистик-дарк тема**: тёмный градиентный фон с пурпуром и золотом
- **Анимированный звёздный фон** на canvas с мерцающими звёздами и поднимающимися частицами
- **Glass-morphism** карточки с подсветкой
- **3D-переворот карт** при открытии
- **Адаптивный mobile-first дизайн**
- Мистические шрифты Cinzel и Cormorant Garamond

## 🚀 Запуск локально

```bash
# Установка зависимостей
bun install

# Запуск dev-сервера
bun run dev

# Сборка для продакшена
bun run build

# Запуск продакшен-сборки
bun start
```

Откройте http://localhost:3000 в браузере.

## 🎴 О Таро

Таро — это зеркало души, а не предсказание будущего. Каждая карта — архетип, живущий в коллективном бессознательном. Когда вы тянете карту, вы активируете этот архетип внутри себя и получаете возможность увидеть то, что раньше было скрыто.

**Старшие арканы** (22 карты) описывают великий путь души от невинности Шута до целостности Мира.

**Младшие арканы** (56 карт) показывают повседневные ритмы через четыре стихии:
- 🜂 **Жезлы** (Огонь) — воля, страсть, творчество
- 🜄 **Кубки** (Вода) — эмоции, чувства, отношения
- 🜁 **Мечи** (Воздух) — мысль, конфликты, решения
- 🜃 **Пентакли** (Земля) — материя, работа, ресурсы
